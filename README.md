# EliminaCode

EliminaCode is an Android application developed in **Kotlin** that focuses on
clean user interaction and efficient handling of application logic.
The project demonstrates mobile development fundamentals using modern Android practices.

## Tech Stack
- **Language:** Kotlin
- **Platform:** Android
- **Architecture:** Activity-based with clear separation of logic
- **Tools:** Android Studio, Gradle, Git

## Features
- Native Android application written in Kotlin
- Clean and readable code structure
- Responsive UI designed for mobile devices
- Focus on application logic and user interaction

## Project Structure
EliminaCode/
├── app/                # Android application module
├── gradle/             # Gradle wrapper files
├── build.gradle        # Build configuration
└── settings.gradle     # Project settings

## Getting Started

### Prerequisites
- Android Studio (latest stable version recommended)
- Android SDK installed

### Run the Application
1. Clone the repository:
```bash
git clone https://github.com/fradom22/Elimina_Code.git
```

2. Open the project in **Android Studio**

3. Let Gradle sync complete

4. Run the app on:
- Android Emulator, or
- Physical Android device
